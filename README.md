# Cosmic Forge

**Cosmic Forge** — это браузерная научно-фантастическая игра-песочница с элементами менеджмента ресурсов, карточных боёв и глубокой кастомизацией корабля.

Главная особенность проекта: симбиоз строгой математической модели (расчёт массы, топлива, урона) и генеративного ИИ (LLM), который отвечает за формирование лора, описание модулей и непредсказуемых боевых ситуаций.

## Особенности игры

- **Бесшовная экономика и полёты**: Каждый килограмм груза на счету. Чем больше трюм забит рудой, тем больше топлива потребуется на обратный путь. Жадность может заставить вас застрять в открытом космосе.
- **Дерево Технологий (Кузня)**: Никаких слепых рецептов. Интерактивная карта чертежей позволяет планировать развитие корабля от ржавой баржи до тяжелого корвета с маскировочными генераторами.
- **Динамическая генерация предметов**: Каждый скрафченный модуль или пушка получает уникальные характеристики, название и циничное описание бортового журнала, сгенерированное нейросетью.
- **Генетика модулей (Мастерская)**: Возможность скрещивать артефакты для получения «Химер». Шанс провала добавляет риска, но позволяет получить предметы с уникальным сочетанием бонусов.
- **Карточные Бои**: Боевая система не полагается на простой автобой. Экипированные модули генерируют вашу «Боевую колоду». Вы маневрируете, разыгрываете карты, используете уязвимости, торгуетесь с пиратами или пугаете их нестабильностью собственного реактора.
- **Хаос-фактор**: Каждая боевая карта содержит скрытое третье действие (Скрытая аномалия). Нейросеть генерирует логическую техническую поломку, вытекающую из ваших действий, добавляя в бой элемент контролируемого хаоса.

## Технический стек

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3. Никаких тяжелых фреймворков. Максимальная производительность и отзывчивость.
- **Backend (BaaS)**: Firebase Realtime Database (сохранение состояния игрока, инвентаря, глобальный рынок модулей).
- **ИИ-генерация**: Интеграция с OpenRouter / Hydra AI (модель Gemini 2.5 Pro) для генерации текстов, эффектов и логики карточных колод.

## Установка и запуск

Проект является полностью клиентским (браузерным) приложением.

1. Клонируйте репозиторий.
2. Откройте файл `index.html` в любом современном браузере.
3. При первом запуске введите `Firebase API Key` (для подключения к БД) и `OpenRouter API Key` (для работы генерации ИИ).
4. Наслаждайтесь игрой!

## Архитектура проекта

- `app.js` — Точка входа, инициализация, навигация.
- `player.js` — Ядро состояния игрока (ресурсы, топливо, эффекты экипировки).
- `mining.js` — Логика экспедиций, расчёт времени, расхода топлива и дропа.
- `forge.js` — Дерево технологий, логика крафта и обращение к LLM.
- `workshop.js` — Механика скрещивания предметов (создание Химер).
- `combat.js` & `combat_deck.js` — Генерация боевых карт на основе модулей.
- `combat_alpha_engine.js` & `combat_sim.js` — Движок карточных боёв и UI.
- `firebase.js` — Обёртка для работы с базой данных.
- `style.css` — Sci-Fi дизайн, анимации и мобильная адаптация.
